> Validates whether a given input is a valid currency symbol.

# Installation
```javascript
npm install is-currency-symbol
```

# Usage
```javascript
const isCurrencySymbol = require('is-currency-symbol');

isCurrencySymbol('$'); // true
isCurrencySymbol('#'); // false
```
